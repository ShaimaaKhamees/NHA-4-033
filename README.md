# SauceDemo Automation Testing Project

This project is an automation testing project for the SauceDemo website using Selenium WebDriver and TestNG.

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven
* IntelliJ IDEA

## Project Features

* Login Testing
* Product Testing
* Cart Testing
* Checkout Testing
* End-to-End Testing

## Test Scenarios

* Valid Login
* Invalid Login
* Add To Cart
* Remove Product
* Checkout Process
* Confirm Order

This project follows a simple automation testing structure using multiple page classes and organized test scenarios.
